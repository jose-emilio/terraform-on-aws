# Ejemplo avanzado

![Terraform on AWS](../../../../../../../images/lab-banner.svg)


Crea dos buckets: uno para logs y otro para datos críticos con logging de acceso activado.
Demuestra el uso de todas las opciones del módulo: versionado, cifrado y access logging.

## Uso

```bash
terraform init
terraform apply
```

## Limpieza

```bash
terraform destroy
```
