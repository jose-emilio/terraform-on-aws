# Ejemplo básico

![Terraform on AWS](../../../../../../../images/lab-banner.svg)


Crea un bucket S3 con la mínima configuración: solo el nombre y el entorno.
El módulo aplica automáticamente versionado, cifrado y bloqueo de acceso público.

## Uso

```bash
terraform init
terraform apply
```

## Limpieza

```bash
terraform destroy
```
